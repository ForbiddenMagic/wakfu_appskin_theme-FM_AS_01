## Gray App Skin

for Wakfu MMO
___
### Windows Install:
- Put "appSkin" folder, replace files if necessary, into your theme folder:
- C:\Users\ **---USER_NAME---** \AppData\Roaming\zaap\gamesLogs\wakfu\themes\ **---THEME_NAME---** \theme\appSkin
- Apply theme in interface settings
